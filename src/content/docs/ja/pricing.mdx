## Software License (Enterprise Self-Hosted)

オンプレミスまたはプライベートクラウドで Aether Platform を運用する場合、以下のソフトウェアライセンス費用が発生します。
（SaaS版をご利用の場合は、これらは利用料に含まれています）

| License Type         | Unit                | Description                                                            |
| :------------------- | :------------------ | :--------------------------------------------------------------------- |
| **User License**     | Per Seat / Month    | 開発者一人当たりにかかるライセンス費用です。標準サポートが含まれます。 |
| **Platform License** | Per Cluster / Month | 管理プレーン (Control Plane) に対する固定ライセンス費用です。          |
| **Add-on Options**   | -                   | Team RAG, Advanced Security などの追加機能オプションです。             |

> [!TIP]
> **AWSなどのインフラ費用は含まれません**
> 自社環境で運用する場合、AWS/GCPなどのクラウド利用料（ランタイムコスト）が別途発生します。
> 概算は [クラウドコスト試算](/pricing-simulator/) ページで計算できます。

## 機能詳細

### 開発・実行環境

- **共通**: Aether Platform の中核である Cloud IDE (code-server ベース) は制限なく利用可能です。
- **Personal**: 共有リソース上で動作するため、ピーク時にパフォーマンスが変動する可能性があります。
- **Corporate**: 組織専用のハードウェアリソースを割り当て可能です。将来的な GPU インスタンスの導入や、特定のリージョンへの配置など、企業の要件に合わせたカスタマイズに対応します。

- **AI 拡張基盤 (MCP & Vector DB)**:
  - **共通**: AI エージェントは MCP (Model Context Protocol) を通じて外部ツールや知識ベース（Vector DB）と連携し、高度な文脈理解に基づいた支援を行います。
  - **Personal**: 自分専用の設定・データとしてのみ利用可能です。プライバシーが完全に保護された自分だけの AI アシスタントを構築できます。
  - **Corporate**: 組織・チーム全体で MCP サーバーやナレッジを共有可能です。社内ドキュメントやプライベートリポジトリを RAG (検索拡張生成) として活用することも、この基盤を通じて実現します。

### セキュリティとガバナンス

- **DNS Filtering & ウイルススキャン**:
  - **共通**: 強力な多層フィルタリングとリアルタイムのウイルススキャンにより、悪意のあるサイトへの接続やマルウェアの混入を防ぎます。
  - **Corporate**: 組織独自のホワイトリスト/ブラックリストの作成や、スキャン検知時の詳細なインシデントレポートを提供します。
- **Network Connectivity Service (NCS)**:
  - **Corporate**: 社内ネットワークへの VPN 接続や、特定の API エンドポイントへのセキュアな閉域接続などの高度なネットワーク制御をすでに利用可能です。
  - **Personal**: 将来的なリリースを予定しています。
- **ガバナンス (監査ログ)**:
  - **Personal**: 個人の利用量把握とセキュリティのため、インターネット通信ログ、内部通信（AIエージェント、MCP）の履歴、およびキャッシュ利用による通信削減量を記録・提供します。
  - **Corporate**: 上記に加え、開発者の詳細な操作ログ（コマンド履歴やファイル編集履歴など）を組織レベルで収集し、高度なコンプライアンス監査に利用可能です。
