## AIネイティブワークフロー

Aether Cloud IDEにおいて、AIは単なるプラグインではありません。OSそのものです。このガイドでは、統合されたエージェントを活用して迅速な開発を行う方法を示します。

## シナリオ: レガシーコードのリファクタリング

型ヒントのないレガシーPythonサービスを開いているとします。

### ステップ 1: コンテキスト分析

機能クラスをハイライトし、`Cmd+K` (または `Ctrl+K`) を押してエージェントを呼び出します。
**"Explain & Refactor"** を選択します。

### ステップ 2: 計画生成

エージェントは、ワークスペースの `CONTRIBUTING.md` に照らしてコードをレビューし、リファクタリング計画を提案します。

- 型ヒントの追加 (MyPy準拠)。
- 共有ロジックを `utils.py` に抽出。
- Docstings（ドキュメンテーション文字列）の追加。

### ステップ 3: 実行

**Apply** をクリックします。エージェントはファイルを直接編集します。
環境は一時的なものなので、統合ターミナルですぐに変更をテストできます。

```bash
pytest
```

### ステップ 4: コミットメッセージの生成

テストが合格したら、ソース管理タブに移動します。
エージェントは、ファイルの変更に基づいてセマンティックなコミットメッセージをすでに入力しています。
`refactor(auth): add type hints and extractjwt validation`
