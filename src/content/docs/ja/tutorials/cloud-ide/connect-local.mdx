## 前提条件

- ローカルにインストールされたVisual Studio Code。
- Aether Platformアカウント。
- アクティブなワークスペース。

## ステップ 1: ワークスペースの詳細を開く

1. **Aetherコンソール**に移動します。
2. アクティブなワークスペースを選択します（`Running`状態である必要があります）。
3. **Connect**ドロップダウンメニューをクリックします。

## ステップ 2: 接続文字列のコピー

**Local VSCode**タブを選択し、生成されたコマンドをコピーします。次のようになります：

```bash
code --remote ssh-remote+workspace-id@gateway.aetherplatform.com /home/workspace
```

## ステップ 3: 接続

1. ローカルターミナルを開きます。
2. コマンドを貼り付けてEnterキーを押します。
3. VSCodeは、リモートクラウドインスタンスに接続された新しいウィンドウを開きます。

> [!TIP]
> **パフォーマンス**: この接続は当社の**ネットワーク接続サービス**トンネルを使用するため、企業のファイアウォールの背後にいる場合でも機能します。
