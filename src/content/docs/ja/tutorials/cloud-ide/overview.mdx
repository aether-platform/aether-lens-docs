## メインストーリー: AI Ready

Aether Platformは、単にVSCodeをブラウザに移行するだけではありません。AI支援開発のための完璧な生息地を構築することです。

GenAIの時代において、環境は単なるテキストエディタ以上のものである必要があります。AIツールに情報を供給する**コンテキスト認識エンジン**である必要があります。当社のCloud IDEは、最初から「AI Ready」になるように構築されており、現代のAIワークフローに必要な高性能コンピュート、事前インデックス化されたコードベース、およびツール統合を提供します。

## 技術アーキテクチャ

### 一時的なコンテナとGPUアクセラレーション

各Cloud IDEインスタンスは、強力なNVIDIA GPUへのオプションアクセスを備えた隔離されたコンテナ内で実行されます。

- **AI最適化イメージ**: PyTorch、TensorFlow、CUDAドライバ、Pythonデータサイエンススタックがプリロードされています。
- **瞬時の推論**: プライベートで低遅延な検証のために、ワークスペース内で直接ローカルLLM（Llama 3、Mistral）を実行します。
- **永続性**: ワークスペースの状態は、高性能な分散ファイルシステムに永続化されます。

### リソーススケーリング

環境は、ワークロードの要求に基づいて動的にサイズ変更できます。

- **Standard**: 2 vCPU、4GB RAM（Web開発に最適）
- **Performance**: 4 vCPU、8GB RAM（コンパイルの多いタスクに最適）
- **Ultra (AI-Ready)**: 8 vCPU、16GB RAM + NVIDIA T4/A10G（MLトレーニングおよびローカルLLM推論用）

## ローカル接続と高度なユースケース

### ローカルVSCode統合

開発者はブラウザの使用を強制されません。**Remote Tunnels**統合を通じて、ローカルのVSCodeインスタンスをAetherクラウド環境に直接接続できます。

- **シームレスな体験**: コードはクラウドで安全に実行されながら、お気に入りのローカルテーマ、キーバインド、拡張機能を使用できます。
- **低遅延**: 最適化されたSSHベースのトンネルにより、ネイティブのようなタイピング感が得られます。

### ビルドのオフロード

重いコンパイルタスクをクラウドにオフロードすることで、生産性を最大化します。

- **シナリオ**: MacBook Airを使用している開発者が、大規模なC++またはRustプロジェクトをコンパイルする必要がある。
- **ソリューション**: ローカルVSCodeを**Ultra**クラスのインスタンスに接続します。重い処理は16コアのクラウドランナーで行われます。

### オンデマンドプロビジョニングによるコスト効率

使用した分だけ支払うことで、クラウド支出を大幅に削減します。

- **アイドルコストゼロ**: 環境は非アクティブ後に自動的にサスペンドされます。コーディングをやめた瞬間に支払いは停止します。
- **適正化（Right-Sizing）**: 開発者は「Standard」インスタンスと「Ultra」インスタンスを数秒で切り替えることができ、高価なGPUリソースは24時間365日ではなく、必要な1時間のトレーニングのため*だけ*に使用できます。
- **TCO削減**: すべての開発者のために高価な常時稼働VDIソリューションやハイエンドのローカルワークステーション（MacBook Pro M3 Max）を用意する必要がなくなります。

## ユーザーストーリー: AIエンジニア

> **シナリオ**: 「AIエンジニアは独自のデータセットでモデルをファインチューニングする必要があるが、ローカルのラップトップにはGPUがない。」

**Aether Platformなし:**

1. IT部門にクラウドインスタンスを申請する（2日待ち）。
2. 素のLinuxボックスにSSH接続する。
3. CUDAドライバのインストールとPythonバージョンの整合に6時間費やす。
4. Jupyter Notebookのポートフォワーディングに苦戦する。

**Aether Platformあり:**

1. "PyTorch Workspace" (Ultra Class) を選択する。
2. **Launch**。
3. 30秒で、完全なVSCode + Jupyter環境の準備が整います。
4. ドライバはプリインストールされています。GPUは準備万端です。
5. エンジニアは `train.ipynb` を開き、"Run" を押します。
