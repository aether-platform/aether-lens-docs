## 前提条件

- Aetherアカウント
- `Node Standard` または `Performance` ワークスペース

## ステップ 1: ワークスペースの作成

1. ダッシュボードから **Next.js** テンプレートを選択します。
2. ワークスペースを起動します。

## ステップ 2: 開発サーバーの起動

テンプレートにはサンプルアプリが付属しています。以下を実行します：

```bash
npm run dev
```

## ステップ 3: プレビューへのアクセス

1. 通知が表示されます: "Application available on port 3000"。
2. **Open in Browser** をクリックします。
3. 固有のプレビューURLが表示されます。

## ステップ 4: ホットリローディングの検証

1. `app/page.tsx` を開きます。
2. 見出しのテキストを変更します。
3. ページの更新なしでプレビューウィンドウが即座に更新されることを確認します。

> [!NOTE]
> **トラブルシューティング**: HMRが失敗する場合は、企業のファイアウォールが `*.preview.aetherplatform.com` へのWebSocket (WSS) 接続を許可していることを確認してください。
