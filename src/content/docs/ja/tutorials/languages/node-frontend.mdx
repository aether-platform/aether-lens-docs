## スタック

Aetherは、React、Next.js、Vueなどのフレームワークに最適化された、最新のJavaScript/TypeScript開発のためのターボチャージされた環境を提供します。

### アーキテクチャ

- **NPM/Yarnキャッシュ**: システム全体で共有されるキャッシュにより、`npm install` が大幅に高速化されます。
- **プレビューURL**: アプリケーションによって開かれたすべてのポート（例: :3000）は、安全で固有のHTTPS URL（例: `https://3000-txn-123.preview.aetherplatform.com`）を介して自動的に公開されます。
- **ホットリローディングサポート**: WebSocketトンネリングが事前構成されており、HMR（Hot Module Replacement）がインターネット経由でシームレスに機能することを保証します。

## ユーザーストーリー: フルスタック開発者

> 「PM（プロジェクトマネージャー）と作業のプレビューを共有したい。」

**従来の方法**: gitにコミットし、CI/CDがステージングにデプロイするのを待ち、タイプミスに気づき、繰り返す。(1サイクル15分)。
**Aetherの方法**: 開発サーバーを起動する。**プレビューURL**をコピーする。SlackでPMに送信する。(10秒)。
