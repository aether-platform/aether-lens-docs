## Accelerate Your Development

"Code changes shouldn't just run tests; they should reveal insights."

開発のスピードを左右するのは、「確認待ち」の時間です。Aether Lens はそれらを最小化し、あなたが最もクリエイティブな瞬間に留まれるようにします。

### ⚡️ 次世代のフィードバック・ループ

1. **Focus on Quality**: AIがリグレッションを監視し、自信を持ってプッシュできます。
2. **Context-Aware**: あなたのプロジェクト特有のルールや「雰囲気」を学習し、適切なアドバイスを行います。
3. **Zero Configuration**: `aether-lens init` と `aether-lens --watch`。それだけで、プロジェクトが知性を持ち始めます。

## 始めましょう

Aether Lens で、あなたのプロジェクトを次のレベルへ。

  <Card title="Quick Start" icon="rocket">
    **事前準備**: OpenAI / Anthropic / Gemini いずれかの API キーが必要です。

    ```bash
    # Install & Run (Nightly)
    uvx --from git+https://github.com/aether-platform/aether-lens aether-lens-cli init
    uvx --from git+https://github.com/aether-platform/aether-lens aether-lens-cli . --watch
    ```
  </Card>
