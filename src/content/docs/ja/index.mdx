## Vibe Test Insights (VTI)

Aether Lens は、単なるテスト自動化ツールではありません。開発者の「バイブス」を理解し、サイクルを極限まで短縮するための知性です。

## Accelerate Your Development

"Code changes shouldn't just run tests; they should reveal insights."

開発のスピードを左右するのは、「確認待ち」の時間です。Aether Lens はそれらを最小化し、あなたが最もクリエイティブな瞬間に留まれるようにします。

### ⚡️ 次世代のフィードバック・ループ (Lens Loop)

Aether Lens の真髄は、**MCP (Model Context Protocol) による AI エージェントとの高度な連携**にあります。開発者が意識することなく、システムの裏側で検証が自律的に繰り返されます。

1. **NPU Accelerated Edge AI**: ローカルの NPU を最大限に活用。生のログやデータをクラウドに送る前に NPU で一次解析・要約（蒸留）を行うことで、トークン消費を最大 90% 削減し、プライバシーと速度を両立させます。
2. **Shadow Verification**: あなたがコードを書いている背後で、Lens が常に UI やロジックをチェック。実行結果を MCP 経由で即座に AI エージェントへ伝達します。
3. **Layered Context Synergy**: 常にすべてのログを AI プロンプトに含める必要はありません。階層化されたコンテキスト提供により、トークンを節約しながら確実な修正インサイトを得られます。
4. **Unified QA Platform**: 開発者が使用するのと同じ環境を QA チームも利用可能。開発からテスト自動化までを同一基盤で行うことで、デリバリー速度を劇的に向上させます。
