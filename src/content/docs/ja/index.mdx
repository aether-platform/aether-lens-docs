## Vibe Test Insights (VTI)

Aether Lens は、単なるテスト自動化ツールではありません。開発者の「バイブス」を理解し、サイクルを極限まで短縮するための知性です。

## Accelerate Your Development

"Code changes shouldn't just run tests; they should reveal insights."

開発のスピードを左右するのは、「確認待ち」の時間です。Aether Lens はそれらを最小化し、あなたが最もクリエイティブな瞬間に留まれるようにします。

### ⚡️ 次世代のフィードバック・ループ

Aether Lens の真髄は、**Vibe Coding Agent との高度な連携**にあります。AI エージェントが書いたコードを即座に検証し、その結果をインサイトとしてフィードバックすることで、開発サイクルを完全に自律化します。

1. **Integrated UI Insights**: あなたが Vibe Coding を行っている背後で、Lens が常に UI をチェック。実行結果の要約を即座に提示するため、そのインサイトを次の Vibe Coding に即座に反映させることができます。
2. **Context-Aware Insights**: 表面的なエラーだけでなく、プロジェクトの「雰囲気」や設計思想に照らした修正アドバイスを提供します。
3. **Unified QA Platform**: 開発者が使用するのと同じ環境を QA チームも利用可能。開発からテスト自動化までを同一基盤で行うことで、デリバリー速度を劇的に向上させます。
4. **Zero Configuration**: `aether-lens init` するだけで、あなたの AI エージェントは「目」を持ち、品質に責任を持つパートナーへと進化します。

## 始めましょう

Aether Lens で、あなたのプロジェクトを次のレベルへ。

  <Card title="Quick Start" icon="rocket">
    **事前準備**: OpenAI / Anthropic / Gemini いずれかの API キーが必要です。

    ```bash
    # Install & Run (Nightly)
    uvx --from git+https://github.com/aether-platform/aether-lens aether-lens-cli init
    uvx --from git+https://github.com/aether-platform/aether-lens aether-lens-cli . --watch
    ```
